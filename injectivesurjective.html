Injective (one-to-one) Functions

The Layman's Definition

The easiest way to remember the definition of an injective (or one-to-one) function, as it is taught in a non-rigourous way, is as follows: Definition (Layman) A function \( f: X \to Y \) is injective (or one-to-one) if it passes the horizontal line/ruler test. If you are new to this idea, the horizontal line/ruler test is as follows: Slide a ruler or horizontal line along the graph. The Mathematical Definition

If you care to read on, what I want to do now is to introduce to you the mathematical definition of an injective function, and then from there, work towards the layman's definition initially introduced. Definition (Mathematical) A function \( f: X \to Y \) is injective (or one-to-one) where for some \( a, b \in X \): if \( f(a) = f(b) \), then \( a = b \). At first glance this seems like a very odd and untuitive definition, but it's a little easier to understand if we recognise that we can simply write \( (x,y) \) co-ordinates as \( (x, f(x)) \). For some, the following definition may be easier to understand: A function \( f: X \to Y \) is injective (or one-to-one) where for some \( x_1, x_2 \in X\), \( y_1, y_2 \in Y \), where \( f(x_1) = y_1 \) and \( f(x_2) = y_2 \) : if \( y_1 = y_2 \), then \( x_1 = x_2 \). where it is assumed that we are working with any two given co-ordinates \( (x_1, y_1), (x_2, y_2) \). Let's simply this a little further. A function \( f: X \to Y \) is injective (or one-to-one) where if two \(y\)-values are the same, then their corresponding \(x\) values are the same.
